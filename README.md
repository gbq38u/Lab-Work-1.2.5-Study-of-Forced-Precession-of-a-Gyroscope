# Lab-Work-1.2.5-Study-of-Forced-Precession-of-a-Gyroscope
В работе исследуется движение гироскопа при действии внешнего момента сил. Экспериментально измеряется зависимость угловой скорости регулярной прецессии от приложенного момента, определяется момент инерции ротора методом крутильных колебаний и частота его вращения.
